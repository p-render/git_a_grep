Getting a Grip on Code

Or is it 'Gitting a Grep?' What is a 'grep'? It's a Unix command to help you find things in files. When I get stuck, it's what I feel I am doing everytime I search the web for help.

This is a collection of web dev related services and libraries that I hope to work through. I made this website for finding practice in coding and for future coding bootcamp preppers. I borrowed the code from Team Treehouse and their tutorials. Check it out! And go to GitHub if you wonder what 'git' is.

I needed a place throw all my disparate bookmarks and notes on web development. This is where I put everything I want to remember. Ok, not everything.

Top 4 Tutorial Websites

Below are four tutorial websites worth looking into. Only Cocecademy is free.

  • Treehouse

  • Codecademy

  • CodeSchool

  • Lynda.com

If you're just starting out, and need conceptual help as well as just learning how to code, then Treehouse is your best bet. If learning to code is priority one above finding a project to work on, Codecademy might be right. And once you feel you can move quickly and want to fill in gaps, then go to Code School. Each of these tutorials has in-browser code challenges as part of their learning process.

If you wanted something where you watch a video while actually doing a programing or software activity, consider what's available at Lynda.com. Their 'Essential Training' series can be useful. I found the Ruby on Rails class to be helpful. Finally, if you are Ruby on Rails focused or are looking to prep for all the front end development code, the Flatiron School has a in depth prework site for their incoming students, with estimation of hours to complete and reasons for why they want you to know it.

Useful Web Development Tools

This list has useful web sites, applications, short cuts, and other useful items to save time.

Web Development in General

  • Adaptive Images - JavaScript image resizer for different screen sizes.
  • Animations - List of Animatible CSS properties.
  • Aptana Studio - an open source IDE. With support for HTML5, CSS3, JavaScript, Ruby, Rails, PHP and Python.
  • Basehold.it - JavaScript baseline overlay for checking vertical rhythm in headings and paragraph spacing.
  • Bootstrap - Helpful social CSS, etc.
  • Can I use… - Compatibility tables for support of HTML5, CSS3, SVG and more in desktop and mobile browsers.
  • Ceaser - CSS Easing Animation Tool
  • Chrome Dev Tools - Information on Chrome's built-in developer tools.
  • CodeKit - Full featured website builder.
  • Codepen - HTML, CSS, JavaScript editor in your browser.
  • Color Picker - CSS Easing Animation Tool
  • Colorzilla Gradient Generator - CSS Easing Animation Tool
  • CSS Button Generator - Settings interface with code output. See also their responsive jQuery content slider bxslider.
  • CSS-tricks - CSS expertise and blog posts
  • Cubic Bezier - Moving on a curve…
  • Cyberduck - FTP and more.
  • Eclipse - an open source IDE. A foundation used by others (Aptana, PyDev).
  • Entities Listed (W3C) - List of entities and their code equivalents.
  • Firebug - Firefox web development tools.
  • Fit Text - jQuery plugins for flexible, scalabe type.
  • FitVids.js - jQuery plugin for fluid video width embeds.
  • Font Deck - Font Collection
  • Fonts.com - Font Collection
  • Foundation - A free, professional front end web development framework.
  • Font Squirrel - Font Collection
  • Google Web Fonts - Font Collection
  • Gridulator - Make quick pixel grids.
  • IcoMoon - Icon Font services.
  • League of Movable Type - Font Collection
  • Lettering.js - jQuery plugin for flexible, scalabe type.
  • MAMP - Local web server for testing sites.
  • Media Queries - Library of examples for media query success.
  • Mobile First - A book and design strategy putting mobile design needs first in the process and scaling up from there.
  • Normalize.css - Browsers render all elements more consistently and in line with modern standards.
  • Pictos - Icon Font services.
  • Picturefill - Picture sizer for different screen sizes.
  • Quirk Tools - Design and display tools. Wireframes, maps, screen testing.
  • Sass - Extension language for CSS, with a sassy attitude.
  • Scout - Scout is a cross-platform app that delivers the power of Sass and Compass into the hands of web designers.
  • Style Tiles - Design layout tool for client review.
  • Sublime Text - sophisticated text editor for code, markup and prose.
  • Subtle Patterns - Browse background patterns and use their photoshop plugin.
  • Symbolset - Icon Font services.

  • This is Responsive - Patterns, resources, and news for creating responsive web experiences.

Python Development Tools

  • Python, the Language - The Mothership. I'm not sure what Python doesn't do.
  • Django - Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design.
  • Pylons Project - Founded by the people behind the Pylons web framework to develop web application framework technology in Python. Rather than focusing on a single web framework, the Pylons Project will develop a collection of related technologies. The first package is the Pyramid web framework.
  • Flask - Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions.
  • Kivy - FOpen source Python library for rapid development of applications that make use of innovative user interfaces, such as multi-touch apps. Kivy runs on Linux, Windows, OS X, Android and iOS.
  • Pandas - Fan open source, BSD-licensed library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language.
  • PyDev - a Python IDE for Eclipse, which may be used in Python, Jython and IronPython development.

Ruby on Rails Development Tools

  • Ruby, the Language - A dynamic, open source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write.
  • Rails Installer - Rails Installer for Mac or Windows. In particular, it installs: Ruby 1.9.3, Rails 3.2, Bundler, Git, Sqlite, osx-gcc-installer, JewelryBox, RVM, SM Framework (11/7/13).
  • Ruby Gems - the Ruby community's gem hosting service.
  • Simple Form - Rails form builder, with Bootstrap styles.
  • Devise - User authentication system.

Javascript Development Tools

  • Mozilla Developer Network’s JavaScript Reference - and where to find their other useful tools at Mozilla.
  • Node.js - Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.
  • Backbone.js - gives structure to web applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing API over a RESTful JSON interface.
  • jQuery - a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
  • Modernizr - a JavaScript library that detects HTML5 and CSS3 features in the user’s browser.

Open Source Tools

It seems to be a golden age for open source projects. These are just a few I am curious about or actually have used.

  • Processing 2 - Processing is a programming language, development environment, and online community, focused on visual arts.
  • Arduino - Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It's intended for artists, designers, hobbyists and anyone interested in creating interactive objects or environments.
  • Raspberry Pi - A $35 Linux based computer board "for your kids to learn about computers." Looks more like the Makers have found it to there liking.
  • Gearman - provides a generic application framework to farm out work to other machines or processes that are better suited to do the work. …it is the nervous system for how distributed processing communicates.
  • Memcached - an in-memory key-value store for small chunks of arbitrary data (strings, objects) from results of database calls, API calls, or page rendering.
  • Sphinx - an open source full text search server. Sphinx lets you either batch index and search data stored in an SQL database, NoSQL storage, or just files quickly and easily — or index and search data on the fly, working with Sphinx pretty much as with a database server.
