Top 3 Tutorial Websites

Below are three tutorial websites worth looking into. Only Cocecademy is free.

  • Treehouse

  • Codecademy

  • CodeSchool

If you're just starting out, and need conceptual help as well as just learning how to code, then Treehouse is your best bet. If learning to code is priority one, and then you'll find a project to work on, then Codecademy might be right. And once you feel you can move quickly and want to fill in gaps, then go to Code School.

Each of these tutorials has in-browser code challenges as part of their learning process. If you wanted something where you watch a video while actually doing a programing or software activity, consider what's available at Lynda.com. Their 'Essiential Training' series can be useful. I found the Ruby on Rails class to be helpful.
